# ONEVA
ONEVA stands for online national election voting app
it is a simple software which enables to automate the process of voting especially in rural areas
it has a simple server and client socket connection with a cross platform kivy gui the socket
connection enables the counting votes where as the counter is the one which is the server
so far the project file uploaded has no socket script added to it its just a basic-UI
but in a near future it shall be added
